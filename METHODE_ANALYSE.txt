CONVENTION D'ANGLE (règle Weapon, PRIME sur tout)
  180 = 0 = tôle à plat  |  90 = équerre  |  45 = pli aigu (fermé/serré)
  L'angle noté = angle de PLI (de combien on plie depuis le plat), PAS l'angle intérieur
  géométrique. Dans le code : la tôle tourne de 180 - AngleCible.
  ⚠️ Commentaire "angle intérieur" de Piece.cs ligne ~20 TROMPEUR, à corriger.
  (Vérifié : BenDFM utilise la même convention — 90=équerre, 45=aigu.)

ORDRE D'ANALYSE D'UN PROFIL — toujours, ne jamais sauter une étape :
  1. LES FACES d'abord. Lire sur le dessin, pli par pli, de quel côté ça ferme.
     La face DÉCOULE de la géométrie, elle ne se choisit pas :
       - un Z alterne les faces (jamais 2 plis de suite même face)
       - un profil qui serpente (bosse puis creux) bascule de face à chaque inversion
       - ne poser [0,0,0,...] QUE si le dessin montre tout du même côté (L, cornière)
     Ne JAMAIS présenter une lecture au solveur sans avoir vérifié les faces au banc.
  2. LES COTES (pans, bleu sur les croquis Weapon), lèvre à lèvre.
  3. LES ANGLES (rouge). Plancher : angle < angle du vé = impossible en l'air.

RÈGLES MÉTIER DURES (codées dans le solveur/détecteur) :
  - PLI FERMÉ EN PREMIER (confirmé sur TOUTE la prod TolTem). Le pli le plus aigu de la
    pièce est TOUJOURS le 1er geste : il rigidifie la tôle ; après on ne passe plus / ça
    tape le tablier / on casse tôle + machine. Contrainte DURE sur le 1er pli SEULEMENT —
    PAS un ordre décroissant sur toute la séquence (casserait la couvertine : 88 suit 92).
    -> le pupitre doit ALERTER si l'op change le pli n°1. (à coder dans la vue)
  - COLLISION TABLIER (le HAUT mobile : coulisseau+inter+poinçon). Garde du bout du bec au
    tablier = 280 mm (Loire Safe), débord bâti 66 mm/côté, marge flexion 5 mm. Hauteur
    d'aile VRAIE = L·sin(180-angle), PAS la projection du repère bissectrice (qui pivote
    avec le pli). Aile 300 à 90° monte à 300 -> tape ; à 45° monte à 212 -> passe.
  - AILE MINI = max(0,63×V, butée mini). Testée sur les DEUX pans (formage ET butée).
  - PLANCHER D'ANGLE = angle du vé. 2045 : V16@45° -> plancher 45°. 4 voies : vés 85/88°
    -> plancher 85° (pli quasi-droit UNIQUEMENT). AUCUNE matrice actuelle ne fait <45°.
  - Le bas (bâti + support matrice) est FIXE = référence y=0 sur la face matrice.

VOCABULAIRE (corrigé par Weapon) :
  tablier = le HAUT mobile. bâti+support matrice = le BAS fixe.
  "inter" = support poinçon = porte-poinçon dans le code. raidisseur = pli à 45°.
  pli écrasé (lèvre rabattue à fond) = formé AUTREMENT (Jouanel), hors périmètre plieuse.
